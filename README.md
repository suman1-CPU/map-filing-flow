# Filing Flow

A shared team kanban for accounting filings — Accounts, Confirmation Statements, VAT and Self Assessment. Jobs flow **Queue → Prerequisites → In Progress → Check & Submit**, with per-person work-in-progress limits, board-wide blockers, and an audited deadline-change trail.

The board data lives in a **private Google Sheet**; this page is only the front-end. On first open you paste the web-app link and access code your admin shares, and enter your name — all stored on your device. No secrets, staff details, or client data are held in this repository.
